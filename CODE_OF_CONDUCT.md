# Code of Conduct

This project adopts the **Contributor Covenant v2.1** verbatim. Full text and
translations: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

In short — we are committed to a harassment-free experience for everyone, regardless
of personal characteristics. The same Code applies in every project space
(repository, issues, pull requests, discussions, in-person events).

## Reporting

Report violations privately to the maintainers via the GitHub Security Advisory
form, or by email to the contact listed in
[reporters's MAINTAINERS.md](https://github.com/cockroachnetworkorg/reporters/blob/main/MAINTAINERS.md).
Reports are handled confidentially.

## Enforcement

Maintainers follow the enforcement guidelines from Contributor Covenant v2.1
(<https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines>):
correction → warning → temporary ban → permanent ban, applied proportionally.
